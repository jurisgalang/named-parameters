Overview

When mixed-in, this module adds hasnamedparameters class method, which can be used to declare methods that are supposed to accepts named-parameters.

Sample usage:

class FooBar include NamedParameters

has_named_parameters :initialize, :required => [ :x ]
def initialize opts = { }
  ...
end

end

FooBar.new :x => '...' # instantiates FooBar FooBar.new :y => '...' # ArgumentError - since :x was not specified
